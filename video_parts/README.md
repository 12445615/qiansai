# Competition Demo Video

The original demo video is split into four files because GitHub rejects regular files larger than 100 MB and this repository has exceeded its Git LFS budget.

To restore the video on Windows PowerShell or Command Prompt, run this command from the repository root:

```cmd
copy /b video_parts\embedded_competition_video.mp4.part01+video_parts\embedded_competition_video.mp4.part02+video_parts\embedded_competition_video.mp4.part03+video_parts\embedded_competition_video.mp4.part04 embedded_competition_video.mp4
```

Original file name: `嵌入式大赛.mp4`

Original size: `314393317` bytes
