# 基于 RK3588 的多模态智能安全联锁系统

本仓库用于集中展示比赛项目代码，代码按模块拆分为 RK3588、Qt 上位机和 STM32 下位机三部分。

- `qt/`：来自 `https://github.com/12445615/qt` 的 `backup-20260626-qt` 备份点，对应提交 `2c26b35`
- `stm32/`：来自本地 STM32 工程 `E:\Linux\airk3588_ai\rk`

## 项目简介

本项目面向危化品库区临时动火检修等高风险作业场景，设计并实现了一套基于 RK3588 与 STM32 协同的工业视觉安全监控与联锁平台。系统采用“边缘智能识别 + 下位机安全联锁”双层架构，实现人员检测、PPE 准入核验、双工作区识别、人员干扰判断、火灾融合检测、分区断电联锁、语音报警、视频留存和云端追溯。

## RK3588 端

RK3588 负责 V4L2 视频采集、RKNN 目标检测、双工作区识别、人员与 PPE 判断、火灾视觉识别、RTMP 推流、H.264 硬件编码、事件录像分片、断网缓存补发、MQTT 云端同步，以及与 STM32 的串口联锁通信。

## Qt 上位机

Qt 上位机用于集中监控现场状态，支持环境监测、工作区电源状态显示、报警状态显示、火灾置信度曲线、最近报警记录、视频预览和历史录像回放。

## STM32 下位机

STM32 负责环境传感、复位按键采集、工作区电源控制、风扇控制和安全联锁执行。当 RK3588 离线或串口通信异常时，STM32 会进入安全保护状态，关闭相关执行机构。

## 核心功能

- 双红色工作区动态识别
- 单人准入上电
- 多人干扰断电
- 区内 PPE 异常分区断电
- 区外 PPE 异常仅告警
- 视觉火焰与环境异常融合确认火灾
- 火灾全局断电与风扇关闭
- RTMP 推流与事件录像
- 断网缓存与恢复后补传
- MQTT 云端同步与 Qt 上位机展示
